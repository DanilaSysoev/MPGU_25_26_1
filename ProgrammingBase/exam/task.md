### Задача

Напишите программу на Python, которая принимает список строковых значений и словарь. Программа должна сделать следующее:

1. Проверить наличие каждого элемента списка в значениях словаря.
2. Если элемент найден, вывести строку вида `"Значение '{элемент}' найдено в словаре."`.
3. Если элемент не найден, вывести строку вида `"Значение '{элемент}' отсутствует в словаре."`

Дополнительно программа должна содержать две функции:
- `check_in_dict(list_values, dict_data)` — основная функция проверки наличия элементов списка в значениях словаря.
- `print_result(result_list)` — вспомогательная функция вывода результатов.

Пример ввода:
```python
my_list = ["яблоко", "груша", "персик"]
my_dict = {"фрукты": ["яблоко", "банан"], "овощи": ["морковь"]}
```

Пример вывода программы:
```
Значение 'яблоко' найдено в словаре.
Значение 'груша' отсутствует в словаре.
Значение 'персик' отсутствует в словаре.
```
