# Домашняя работа 3 - цикл с предусловием (дедлайн 25.10.25 включительно, до 10 баллов)

---

## Критерии оценивания

---

Полный балл за задачу ставится, если решение полностью соответствует условию и на входных данных заданного формата выводит верный ответ в заданном формате.

**Штрафы (за каждую задачу)**
- незначительные ошибки в алгоритме (граничные условия, отдельные случаи) - **0.5 балла**
- неверный формат ввода/вывода - **0.5 балла**
- значительные ошибки в алгоритме (неверно обрабатывается целый класс входных данных) - **1.5 балла**
- критические ошибки в алгоритме (неверно обрабатывается несколько классов входных данных) - **все баллы**

**Штрафы (за общее решение)**
- незначительные огрехи стиля (число пробелов в отступе, неудачно расставленные скобки) - **10% с округлением до 0.5 в большую сторону**
- значительные огрехи стиля (имена переменных в стиле `a`, `b`, `c`, строки кода, не умещающиеся в экран) - **30% с округлением до 0.5 в большую сторону**

## Блок 1. Базовое использование цикла `while` **(1 балл)**

---

### 1. Счёт до заданного числа

Напишите программу, которая получает число `N` и выводит все числа от `1` до `N` включительно, каждое на новой строке.

**Формат входных данных**
Одно целое число `N` (`1 ≤ N ≤ 10^6`).

**Формат выходных данных**
`N` строк, в каждой — очередное число от `1` до `N`.

**Пример**
Ввод:

```
5
```

Вывод:

```
1
2
3
4
5
```

---

### 2. Обратный отсчёт

Напишите программу, которая получает число `N` и выводит числа от `N` до `1` в порядке убывания.

**Формат входных данных**
Одно целое число `N` (`1 ≤ N ≤ 10^6`).

**Формат выходных данных**
`N` строк, в каждой — очередное число от `N` до `1`.

**Пример**
Ввод:

```
4
```

Вывод:

```
4
3
2
1
```

---

### 3. Вывод чётных чисел

Напишите программу, которая получает число `N` и выводит все чётные числа от `2` до `N` включительно.

**Формат входных данных**
Одно целое число `N` (`2 ≤ N ≤ 10^6`).

**Формат выходных данных**
Все чётные числа от `2` до `N`, каждое на новой строке.

**Пример**
Ввод:

```
7
```

Вывод:

```
2
4
6
```

---

### 4. Степени двойки

Напишите программу, которая получает число `N` и выводит все степени двойки, не превышающие `N`.

**Формат входных данных**
Одно целое число `N` (`1 ≤ N ≤ 10^9`).

**Формат выходных данных**
Все числа вида `2^k`, не превышающие `N`, каждое на новой строке.

**Пример**
Ввод:

```
20
```

Вывод:

```
1
2
4
8
16
```

## Блок 2. Работа с последовательностями и вводом  **(2 балла)**

---

### 5. Сумма последовательности

Напишите программу, которая считывает последовательность положительных целых чисел и вычисляет их сумму. Ввод числа `0` означает конец последовательности.

**Формат входных данных**
Несколько строк, в каждой одно целое число. Последнее введённое число всегда `0`.

**Формат выходных данных**
Одно целое число — сумма всех введённых положительных чисел.

**Пример**
Ввод:

```
120
250
90
0
```

Вывод:

```
460
```

---

### 6. Подсчёт положительных чисел

Напишите программу, которая считывает последовательность чисел и подсчитывает, сколько среди них положительных. Ввод числа `0` завершает последовательность.

**Формат входных данных**
Несколько строк, в каждой одно целое число. Последнее число — `0`.

**Формат выходных данных**
Одно целое число — количество положительных чисел во введённой последовательности.

**Пример**
Ввод:

```
5
-3
12
8
0
```

Вывод:

```
3
```

### 7. Минимум последовательности

Напишите программу, которая считывает последовательность целых чисел и находит минимальное значение. Ввод числа `0` завершает последовательность.

**Формат входных данных**
Несколько строк, в каждой одно целое число. Последнее число — `0`.

**Формат выходных данных**
Одно целое число — минимальное значение среди всех введённых чисел (не считая `0`).

**Пример**
Ввод:

```
8
4
-2
7
0
```

Вывод:

```
-2
```

### 8. Среднее значение

Напишите программу, которая считывает последовательность целых чисел и вычисляет их среднее арифметическое. Ввод `0` завершает последовательность.

**Формат входных данных**
Несколько строк, в каждой одно целое число. Последнее число — `0`.

**Формат выходных данных**
Одно вещественное число — среднее значение введённых чисел, округлённое до двух знаков после запятой.

**Пример**
Ввод:

```
5
10
15
0
```

Вывод:

```
10.00
```

## Блок 3. Условия и конструкция `while-else` **(2 балла)**

---

### 9. Поиск первого кратного 7

Напишите программу, которая считывает целые числа одно за другим, пока не встретится число `0`.
Если среди введённых чисел встречается хотя бы одно, кратное 7, выведите первое из них.
Если таких чисел нет, выведите `NOT FOUND`.
Используйте конструкцию `while` и `while-else`.

**Формат входных данных**
Несколько строк, в каждой одно целое число. Последнее число — `0`.

**Формат выходных данных**
Если найдено кратное 7 число, вывести строку:

```
FOUND <число>
```

Если не найдено — вывести строку:

```
NOT FOUND
```

**Пример 1**
Ввод:

```
5
14
9
0
```

Вывод:

```
FOUND 14
```

**Пример 2**
Ввод:

```
2
3
5
9
0
```

Вывод:

```
NOT FOUND
```

### 13. Таблица умножения

Напишите программу, которая выводит таблицу умножения размером `N × N` с помощью вложенных циклов `while`.
В ячейке с координатами `(i, j)` должно находиться произведение чисел `i` и `j`.

**Формат входных данных**
Одно целое число `N` (`1 ≤ N ≤ 9`).

**Формат выходных данных**
`N` строк, каждая из которых содержит `N` чисел — произведения `i × j`, разделённые пробелом.

**Пример**
Ввод:

```
3
```

Вывод:

```
1 2 3
2 4 6
3 6 9
```

### 14. Прямоугольник из символов

Напишите программу, которая выводит прямоугольник из символов `#` размером `H × W`.
Используйте вложенные циклы `while`.

**Формат входных данных**
Два целых числа `H` и `W` (`1 ≤ H, W ≤ 20`), разделённые пробелом.

**Формат выходных данных**
Прямоугольник из `#` высоты `H` и ширины `W`.

**Пример**
Ввод:

```
3 5
```

Вывод:

```
#####
#####
#####
```

### 15. Прямоугольная рамка

Напишите программу, которая выводит прямоугольник высоты `H` и ширины `W`.
Границы заполнить символом `#`, внутреннюю область — точками `.`.
Используйте вложенные циклы `while`.

**Формат входных данных**
Два целых числа `H` и `W` (`2 ≤ H, W ≤ 20`), разделённые пробелом.

**Формат выходных данных**
`H` строк длины `W`, описывающих прямоугольную рамку.

**Пример**
Ввод:

```
4 7
```

Вывод:

```
#######
#.....#
#.....#
#######
```
