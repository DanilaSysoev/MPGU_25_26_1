# Задание: Проект на свободную тему

## Описание

Вам нужно разработать программу на свободную тему. Это может быть игра, приложение по управлению финансами, личный дневник или что-то еще.

## Формат сдачи

В качестве результата вы должны прислать zip архив с реешнием. архив, помимо файлов кода, должен содержать README с описанием проекта, инструкцией по запуску и желаемым баллом за проект.
